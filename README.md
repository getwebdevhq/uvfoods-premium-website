# UV Foods — Premium Corporate Website (V2)

A cinematic, sustainability-focused premium corporate website for **UV Foods Private Limited**, a modern nutrition infrastructure company serving institutional and government-scale food programs across India. 

Operating since 1986, UV Foods fights hunger and malnutrition through large-scale nutritional manufacturing, supply chains covering 600+ districts, and direct agricultural cooperative partnerships.

---

## 🎨 Premium Editorial Design Direction

This platform has been transformed from a generic layout into a global-caliber environmental and agri-tech portal:
- **Color Palette**: Clean cream background (`#F7F5F0`), deep forest green (`#1B2A1F`), olive accents (`#314D38`), and vibrant lime highlights (`#C7F36B`).
- **Typography**: Clash Display headings paired with Inter body text.
- **Micro-Animations**: Slow, cinematic page entry transitions powered by **Framer Motion**.
- **Kinetic Scrolling**: Deceleration-based, hardware-accelerated smooth scroll using **Lenis**.
- **Visual Assets**: Immersive, customized photography generated for our facilities, organic farming, and community programs.

---

## 🚀 Key Technical Features

### 1. Multi-Page React Router SPA
Contains 6 structured, responsive pages designed for institutional credibility:
- **Home (`/`)**: High-impact editorial previews, animated counter metrics, and rapid contact access.
- **About (`/about`)**: Generational legacy timeline, quality control philosophies, and mission pillars.
- **Capabilities (`/capabilities`)**: Bullet grids detailing fortification methods, supply logistics, and an end-to-end value chain timeline.
- **Infrastructure (`/infrastructure`)**: 280+ MT capacity dashboard (General, Proprietary, and Fortification lines) and quality timelines.
- **Sustainability (`/sustainability`)**: Five-pillar cooperative agriculture and community health narrative overlays.
- **Contact (`/contact`)**: Secure partnership inquiry desk with client filters and automated confirmation states.

### 2. Search Optimization (SEO) & Structured Schema
- Dynamic Title and OpenGraph meta updating on route changes.
- Automatic **JSON-LD Organization Schema** injection inside the `<head>` of the application.

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- npm / yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/getwebdevhq/uvfoods-premium-website.git
   cd uvfoods-premium-website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
